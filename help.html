How to build & release github pages

1.To install latest nodejs and npm

curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
sudo apt-get install nodejs
sudo npm install npm@latest -g
2.To build github pages with Docusaurus
export GIT_USER=TheBestOrNothing
npm install // in Docusaurus dir
npm run publish-gh-pages //in websit dir
