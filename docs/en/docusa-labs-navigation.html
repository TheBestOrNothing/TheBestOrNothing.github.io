Add navigation

To add navigation in the websites, please follow the steps:

  1. Add the navigation structure in website/sidebars.json file.
  "eth": {
    "Ethereum": [
      "eth-01-starting"
    ],
    "Transaction": [
    ],
    "Contracts": [
    ]
  }
  1. Add entry in headerLinks array in website/siteConfig.js file.
  headerLinks: [
    {doc: 'btc-01-starting', label: 'BTC'},
    {doc: 'eth-01-starting', label: 'ETH'},
    ......
  ]

> Notice:"eth-01-starting" is a ID of md file

  1. Add a file which id is "eth-01-starting".

  2. Publish the website by npm run.

    npm run publish-gh-pages
