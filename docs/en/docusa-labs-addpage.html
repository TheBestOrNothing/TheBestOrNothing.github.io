Add new pages

To add new pages in the websites with Docusa, please follow the steps:

  1. Add "docusa-labs-addpage" (the id of the doc) to the LABs section in the website/sidebars.json file
  "docs": {
    "Getting Started": [
      "installation",
      "site-preparation",
      "site-creation",
      "publishing"
    ],
    "Guides": [
      "blog",
      "custom-pages",
      "search",
      "navigation",
      "translation",
      "versioning"
    ],
    "API": [
      "commands",
      "doc-markdown",
      "api-pages",
      "site-config"
    ],
    "LABs": [
      "docusa-labs-publish",
      "docusa-labs-addpage"
    ]
  }

  1. Add md docs with docusa-labs-addpage.

    id: docusa-labs-addpage

    title: Add new pages

  2. Publish the website by npm run.

    npm run publish-gh-pages
