Translate

手动翻译步骤

Publish websites

To translate the websites to Chinese, please follow the steps: 按照以下步骤手动翻译

  1. Create an new md file in website/docs with new id.

  2. Add the md id to website/sidebars.json.

  3. copy the md file in website/translated_docs/zh-CN

    Notice: two md files with same id and title

  4. Adapt the md content in in website/translated_docs/zh-CN.

    Notice: Change the contents of md file, rather than id and title

  5. Run the following code , but not checkin.

    npm run publish-gh-pages

    Notice all the info needed translate is put in website/i18n/en.json file

  6. Copy the content in en.json to zh-CN.json file and Adapt the zh-CN.json in website/i18n dir.

    Notice: Change the contents which match the md id

  "localized-strings": {
    "next": "Next",
    "previous": "Previous",
    "commands": "CLI Commands",
    "doc-markdown": "Markdown Features",
    "api-pages": "Pages and Styles",
    "site-config": "siteConfig.js",
    "btc-01-starting": "比特币之旅",
    "btc-01-starting": "比特币之旅",
    "docusa-labs-translate-manual": "手动翻译",
  1. Publish the website by npm run.

    npm run publish-gh-pages
